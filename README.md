# Password Generator Tool

A clean single-page web utility for creating strong, varied passwords with instant strength feedback and clipboard support.

## Features
- Adjust the length between 8 – 32 characters.
- Toggle uppercase, lowercase, numbers, and symbols independently.
- Color-coded strength meter that updates immediately.
- One-click copy to clipboard with visual confirmation.

## Getting Started
```bash
git clone https://github.com/sunnymendapara15/password-generator-tool.git
cd password-generator-tool/frontend
npm install
npm start
```

The app launches on [http://localhost:3000](http://localhost:3000) by default.

## Testing
- `npm test` (runs the React test watcher).

## Production Build
- `npm run build` bundles the app for deployment.
